
WineTruth

## Install ##

0. Install Git
1. Install VirtualBox
2. Install Vagrant
3. Open the WineTruth directory in a terminal (Git provides this on Windows).
4. `vagrant up`
5. `vagrant ssh`
6. `cd /synced/main`
7. `make run`

## Test ##

Same instructions, but `make test` from Vagrant instead.
